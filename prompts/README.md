## Как устроен промт в этом репо

Каждая папка содержит:

- **`README.md`** — карточка промта: зачем, для кого, как применить, пример, нюансы
- **`prompt.md`** — чистый текст промта, готов к копипасте (открой через **Raw** → скопировал → вставил в System Instruction)

> Все промты протестированы лично на реальных задачах.
> Если что-то не работает — [открой issue](https://github.com/syrtsovk/Syrtsovk_toolkit/issues), починим.
